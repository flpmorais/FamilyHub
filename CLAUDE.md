# FamilyHub — Vacation Module

## Branch Rule
This worktree is for the **VACATION** module.
- ONLY commit to the `vacation-module` branch.
- NEVER switch branches, merge, or push to main.
- Before any git commit, verify you are on `vacation-module` with `git branch --show-current`.

## Expo Port
Run Expo on port **8081**: `npx expo start --port 8081`

## Module Scope
V1 Vacation module: packing lists, booking tasks, templates, categories, tags, offline sync, M3 theme.
